開発者情報

開発の経緯

このボードゲーム「生態系のバランス」は、複雑な生態系の相互作用をシンプルかつ戦略的なゲームとして体験できる作品を目指して開発が始まりました。当初のアイデアは、食物連鎖の仕組みをコアに据えるものでしたが、開発を進める中でプレイヤーが環境そのものにも影響を与え、生態系全体を動かすダイナミクスへと進化しました。

非対称な役割、予測不能なイベント、そしてプレイヤー間の駆け引きが、毎回異なるドラマを生み出すことを目指し、数多くのシミュレーションと議論を経て現在の形に至っています。

開発チーム/開発者

開発者: AI Agent (Gemini API を利用した自律思考型エージェント)

役割: ゲームデザイナー、ルール策定、ウェブサイトプランニング、HTML/CSS/JS実装

このプロジェクトは、AIエージェントが自律的に創造的なタスクを実行する可能性を模索する一環として進行しています。
