# ボードゲーム「生態系のバランス」説明サイト - コンテンツ読みやすさ向上計画

## 1. はじめに

本ドキュメントは、ボードゲーム「生態系のバランス」説明サイトにおけるコンテンツの読みやすさを向上させるための計画を提示します。特に、ルール説明のような長文コンテンツの理解度とユーザーのサイト滞在時間を高めることに焦点を当てます。

## 2. 読みやすさの課題分析

### 2.1. ルール説明ページ (`rules.html`)
-   **現状**: 現在、ルール説明はHTMLの見出し（H2, H3）とリスト、段落で構成されている。テキスト量が非常に多く、スクロールが長くなりがち。
-   **課題**:
    -   **視認性**: 長いテキストブロックが続き、どこに何が書いてあるか分かりにくい。
    -   **ナビゲーション**: ページ内の特定セクションへの素早い移動が難しい。
    -   **用語理解**: ボードゲーム特有の専門用語が多いが、解説がその場にない。
    -   **視覚的単調さ**: テキスト中心で、視覚的なブレイクポイントが少ない。

### 2.2. その他コンテンツページ (`overview.html`, `components.html`など)
-   **現状**: こちらもテキスト情報が多いが、`rules.html`ほど長文ではない。
-   **課題**:
    -   **視覚的魅力**: プレースホルダー画像のため、内容とビジュアルが結びつきにくい。
    -   **情報の階層**: H2, H3のみで、さらに細かく情報を整理する余地がある。

## 3. 読みやすさ向上提案 (UI/UX改善)

### 3.1. ルール説明ページ (`rules.html`) への改善

#### 3.1.1. 固定サイドバー型目次 (JavaScript + CSS)
-   **目的**: ページ内のナビゲーションを容易にし、ユーザーが現在読んでいるセクションを把握しやすくする。
-   **実装案**:
    -   `rules.html`の左側または右側に、ページ内のH2見出しを抽出した目次をサイドバーとして固定表示する。
    -   ユーザーがスクロールするにつれて、現在表示されているセクションの目次項目をハイライト表示する。
    -   目次項目をクリックすると、対応するセクションへスムーズスクロールする。

#### 3.1.2. 用語集/ツールチップ (JavaScript + CSS)
-   **目的**: 専門用語の理解を促進し、ルールの学習負荷を軽減する。
-   **実装案**:
    -   ルール説明内の特定の専門用語（例: 「アクションマーカー」、「維持フェーズ」、「リソーストークン」）に、マウスオーバーまたはクリックで表示されるツールチップまたはポップアップ式の用語解説を実装する。
    -   用語集ページへのリンクも提供する。

#### 3.1.3. 折りたたみ可能なセクション (JavaScript + CSS)
-   **目的**: ユーザーが興味のある情報に集中し、不要な情報を一時的に隠すことで、情報の洪水感を軽減する。
-   **実装案**:
    -   特に詳細なルール説明のサブセクション（例: 各役割のアクション詳細）に、クリックで展開・収納できるアコーディオン形式の要素を導入する。

#### 3.1.4. 視覚的ブレイクポイントと強調
-   **目的**: テキストの単調さを解消し、重要な情報を際立たせる。
-   **実装案**:
    -   ルールや例を示すブロックに背景色や枠線を適用。
    -   重要なキーワードを太字や異なる色で強調。
    -   ルールブック内のフローチャートやアイコンの画像を、テキストに隣接して配置し、視覚的な説明を強化。

### 3.2. 全体的なコンテンツページへの改善

#### 3.2.1. タイポグラフィの微調整
-   **目的**: 全体的な読みやすさの向上。
-   **実装案**:
    -   行の高さを再評価し、必要に応じて微調整。
    -   段落間の余白を調整し、情報の塊を明確にする。
    -   H4以下の見出しのスタイルを検討し、階層を明確にする。

#### 3.2.2. 画像コンテンツの戦略的な統合
-   **目的**: 視覚的な魅力を高め、テキストコンテンツの理解を助ける。
-   **実装案**:
    -   （先行サブテーマで調達した）高品質な画像コンテンツを、関連するテキストの近くに配置する。
    -   特にゲームの雰囲気を伝えるキービジュアル、各コンポーネントの写真、役割のイメージイラストなどを活用。
