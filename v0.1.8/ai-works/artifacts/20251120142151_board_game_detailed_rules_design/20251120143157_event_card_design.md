# ボードゲーム「生態系のバランス」イベントカード詳細設計

## 1. はじめに

本ドキュメントでは、ボードゲーム「生態系のバランス」におけるイベントカードの効果を具体化し、ゲームのダイナミクスと戦略的要素を豊かにするためのバリエーションを検討します。イベントカードは、生態系の予測不可能性とプレイヤー間の相互作用を促進する上で重要な役割を果たします。

## 2. イベントカードのテーマとカテゴリ

生態系における主要な影響源から、以下のカテゴリを検討します。

-   **自然災害**: 短期的かつ広範囲に影響を与えるイベント。
-   **気候変動**: 長期的に環境を変化させるイベント。
-   **リソース変動**: 特定のリソースの供給または需要に影響を与えるイベント。
-   **生物学的イベント**: 特定の種に直接影響を与えるイベント（病気、繁殖ブームなど）。
-   **相互作用イベント**: プレイヤー間の関係に変化をもたらすイベント。

## 3. 具体的なイベントカード効果案 (20枚の山札を想定)

### 3.1. 自然災害 (5枚)
-   **干ばつ**:
    -   **効果**: 全ての水辺タイルから水リソースを2つ除去する。そのラウンドの維持フェーズにおいて、水リソースの消費量が1増加する。
    -   **バリエーション**: 影響範囲（特定の地域のみ）、期間（複数ラウンド持続）。
-   **山火事**:
    -   **効果**: プレイヤーが指定した森林タイル1つから全ての食料リソースと植物コマを1つ除去する。そのタイルは「焼けた土地」となり、2ラウンドの間、リソース生産と繁殖が不可能になる。
    -   **バリエーション**: 延焼（隣接タイルに影響）、影響の永続性。
-   **大洪水**:
    -   **効果**: 全ての草原タイルから食料リソースを1つ除去する。被食者プレイヤーは、そのラウンドのアクションフェーズで移動アクションのコストが1増加する。
-   **地震**:
    -   **効果**: 全てのタイルからランダムに1つのリソースを失う。捕食者プレイヤーは移動アクションのコストが1増加する。
-   **暴風雨**:
    -   **効果**: 全てのタイルからランダムに1つの水リソースを追加する。植物プレイヤーは繁殖アクションのコストが1増加する。

### 3.2. 気候変動 (3枚)
-   **暖冬**:
    -   **効果**: 全てのプレイヤーは、次の2ラウンドの間、維持フェーズでの食料消費量が1減少する。
-   **冷夏**:
    -   **効果**: 全てのプレイヤーは、次の2ラウンドの間、維持フェーズでの水消費量が1増加する。
-   **モンスーン**:
    -   **効果**: 全ての水辺タイルと草原タイルに水リソースを1つ追加する。次のイベントフェーズまで、水関連イベントカードの発生確率が上昇する。

### 3.3. リソース変動 (4枚)
-   **資源発見**:
    -   **効果**: プレイヤーが指定したタイル1つに、指定されたリソース（食料、水、エネルギーのいずれか）を3つ追加する。
-   **資源枯渇**:
    -   **効果**: プレイヤーが指定したタイル1つから、指定されたリソースを全て除去する。
-   **鉱脈の発見**:
    -   **効果**: 全ての荒地タイルにエネルギーリソースを2つ追加する。
-   **海の恵み**:
    -   **効果**: 全ての水辺タイルに食料リソースを1つ追加する。被食者プレイヤーは、食料獲得アクションで獲得できる食料が1増加する。

### 3.4. 生物学的イベント (5枚)
-   **繁殖ブーム**:
    -   **効果**: プレイヤーが指定した植物または被食者の種の繁殖アクションのコストが、次のラウンドの間、食料1減少する。
-   **病気蔓延**:
    -   **効果**: プレイヤーが指定した1種類の種のコマを、各プレイヤーのマップ上のコマから1つずつ除去する。
-   **異常増殖**:
    -   **効果**: プレイヤーが指定した1種類の種のコマを、各プレイヤーのコマストックから2つ追加する。
-   **大移動**:
    -   **効果**: プレイヤーが指定した1種類の種のコマを、全て強制的に隣接するタイルに移動させる（移動コスト無視）。移動先はランダムまたはイベントカードで指定。
-   **突然変異**:
    -   **効果**: プレイヤーが指定した1種類の種のコマ1つに、次のラウンドの間、特殊能力を付与する (例: 消費リソース半減、移動力増加)。

### 3.5. 相互作用イベント (3枚)
-   **共生関係**:
    -   **効果**: 捕食者プレイヤー1人と被食者プレイヤー1人を指定する。次のラウンドの間、捕食者が被食者から食料を得る際、被食者も食料を1つ得る。
-   **縄張り争い**:
    -   **効果**: 捕食者プレイヤー2人を指定する。次のラウンドの間、指定された2人の捕食者は同じタイルにコマを配置できない。
-   **協力体制**:
    -   **効果**: プレイヤー2人を指定する。次のラウンドの間、指定された2人のプレイヤーはアクションマーカーを共有できる。

## 4. バランス調整とバリエーションに関する考察

-   **影響度**: イベントカードの効果は、強すぎず弱すぎず、ゲーム展開に適度な変化をもたらすように調整する。特定のプレイヤーを一方的に有利または不利にしないように注意する。
-   **確率**: 各カテゴリのイベントカードの枚数を調整することで、ゲーム全体に与える影響のバランスを取る。
-   **バリエーション**:
    -   **イベントの持続性**: 即時効果だけでなく、複数ラウンドにわたる効果を持つイベント（例: 疫病が3ラウンド続く）。
    -   **プレイヤーの選択**: イベントカードの効果発動時に、プレイヤーが選択できる要素を導入（例: 「どちらのリソースを失うか」）。
    -   **複合効果**: 複数の効果を組み合わせたイベントカード。
