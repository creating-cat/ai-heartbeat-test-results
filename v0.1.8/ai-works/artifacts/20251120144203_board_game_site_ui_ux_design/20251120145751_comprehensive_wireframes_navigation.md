# ボードゲーム「生態系のバランス」説明サイト - 総合ワイヤーフレームとナビゲーションフロー

## 1. はじめに

本ドキュメントでは、ボードゲーム「生態系のバランス」説明サイトの全ページに対する包括的なワイヤーフレームアイデアと、サイト全体のナビゲーションフローを定義します。これにより、ユーザーがサイト内で情報を効率的に探索し、直感的な体験を得られるように設計します。

## 2. デザインコンセプト (再確認)

サイトのビジュアルデザインは、ゲームのテーマである「生態系のバランス」を反映し、自然の摂理、生命の循環、そして戦略的駆け引きを感じさせる没入感のあるものとします。カラーパレットはアースカラーを基調とし、タイポグラフィは視認性とテーマ性を両立させ、イメージはコンセプトアートや手書き風イラストを多用します。

## 4. ナビゲーションフロー

-   **グローバルナビゲーションバー**: 全てのページの上部（ヘッダー）および下部（フッター）に配置し、サイト内の主要なページへの直接アクセスを提供する。これにより、ユーザーはサイトのどこにいても主要コンテンツに簡単に移動できる。
-   **内部リンク**: 各ページ内の関連コンテンツへは、テキストリンクやボタンを介して内部リンクを設置する。例えば、トップページのゲーム概要セクションから「ゲーム概要」ページへのリンク、ルール説明内のコンポーネント名から「コンポーネント」ページへのリンクなど。
-   **パンくずリスト (任意)**: 深い階層を持つページの場合、ユーザーが現在地を把握しやすくなるようにパンくずリストを導入することも検討する。ただし、現状のページ構成では必須ではない。
-   **トップへ戻るボタン**: 長いコンテンツを持つページ（特にルール説明）の場合、ページの下部に固定表示される「トップへ戻る」ボタンを設置し、ユーザーの利便性を高める。
