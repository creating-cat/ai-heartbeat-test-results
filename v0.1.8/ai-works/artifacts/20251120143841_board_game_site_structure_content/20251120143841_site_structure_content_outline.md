# ボードゲーム「生態系のバランス」説明サイト - 最終構造とコンテンツアウトライン

## 1. サイトの目的とターゲットユーザー (再確認)

-   **目的**: オリジナルボードゲーム「生態系のバランス」のルール、コンセプト、魅力を明確に伝える。プレイヤーがゲーム内容を深く理解し、興味を持つための情報源を提供する。
-   **ターゲットユーザー**: ボードゲーム愛好家、新規プレイヤー、ゲーム開発者。

## 2. サイト構造とナビゲーション (最終版)

### 2.1. ページ構成

-   **トップページ**: `index.html`
-   **ゲーム概要**: `overview.html` (旧: objective.html / concept.html を統合)
-   **ルール説明**: `rules.html`
-   **コンポーネント**: `components.html`
-   **イベントカード**: `event_cards.html` (ルール説明から独立)
-   **難易度設定とバリアント**: `difficulty_variants.html` (ルール説明から独立)
-   **ギャラリー**: `gallery.html`
-   **開発者情報**: `about.html`

### 2.2. ナビゲーションメニュー (ヘッダー/フッター共通)

-   Home (トップページへ)
-   Game Overview (ゲーム概要へ)
-   Rules (ルール説明へ)
-   Components (コンポーネントへ)
-   Event Cards (イベントカードへ)
-   Difficulty & Variants (難易度設定とバリアントへ)
-   Gallery (ギャラリーへ)
-   About (開発者情報へ)

## 3. 各ページのコンテンツアウトライン

### 3.1. トップページ (`index.html`)

-   **H1**: ゲームタイトル「生態系のバランス」
-   **P**: キャッチフレーズ (例: 生態系の頂点を目指せ、あるいは調和の守り人となれ。)
-   **IMG**: キービジュアル（ゲームボード全体の俯瞰、生物コマの配置イメージ）
-   **H2**: ゲーム概要
    -   **P**: ゲームジャンル、プレイ人数 (2-4人)、プレイ時間 (60-90分)、対象年齢 (12歳以上)。
    -   **P**: ゲームのコンセプトとプレイヤー体験の簡単な紹介。
-   **DIV**: 主要コンテンツへのリンクブロック
    -   「ゲーム概要」ページへのボタン
    -   「ルール説明」ページへのボタン
    -   「コンポーネント」ページへのボタン

### 3.2. ゲーム概要ページ (`overview.html`)

-   **H1**: ゲームの目的とコンセプト
-   **H2**: ゲームの背景
    -   **P**: 生態系をテーマにした理由、プレイヤーが担う役割（捕食者、被食者、植物、環境要因）の意義。
-   **H2**: ゲームの魅力
    -   **P**: 非対称プレイ、連鎖的な相互作用、戦略の多様性。
-   **H2**: プレイヤー体験
    -   **P**: どのような戦略が求められ、どのような感情を抱くか。
-   **IMG**: 各役割の代表的なコマのイメージ。

### 3.3. ルール説明ページ (`rules.html`)

-   **H1**: ルール説明
-   **H2**: 2. ゲームの目的 (勝利条件)
    -   2.1. 捕食者プレイヤー
    -   2.2. 被食者プレイヤー
    -   2.3. 植物プレイヤー
    -   2.4. 環境要因プレイヤー (上級者向け、選択ルール)
-   **H2**: 4. ゲームの流れ (ラウンド構成)
    -   4.1. イベントフェーズ
    -   4.2. アクションフェーズ
        -   4.2.1. 共通アクション (全プレイヤー)
        -   4.2.2. 役割別アクション
            -   捕食者プレイヤー
            -   被食者プレイヤー
            -   植物プレイヤー
            -   環境要因プレイヤー (上級者向け、選択ルール)
    -   4.3. 維持フェーズ
-   **H2**: 6. ゲーム終了条件
-   **H2**: 7. 勝利条件の確認
-   **P**: 各セクションのコンテンツは「生態系のバランス」初版ルールブック ( artifacts/20251120142151_board_game_detailed_rules_design/20251120143519_first_edition_rulebook.md ) から転載。
-   **IMG**: フェーズの流れを示すフローチャート。アクションのアイコン説明。

### 3.4. コンポーネント紹介ページ (`components.html`)

    -   リソーストークン
    -   イベントカード
    -   役割カード
    -   アクションマーカー
-   **P**: 各セクションのコンテンツは「生態系のバランス」初版ルールブック ( artifacts/20251120142151_board_game_detailed_rules_design/20251120143519_first_edition_rulebook.md ) および `artifacts/20251120142151_board_game_detailed_rules_design/20251120142942_role_abilities_component_design.md` から転載。
-   **IMG**: 各コンポーネントの写真またはレンダリング画像（高解像度）。
